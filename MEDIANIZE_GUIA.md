# Guia: Converter Projeto Verão em APK com Medianize

## 📱 O que é Medianize?

Medianize (median.co) é uma plataforma que converte aplicações web em APK nativo Android de forma rápida e fácil, sem necessidade de instalar Android SDK ou Gradle.

## 🚀 Passos para Gerar o APK

### 1. Acessar o Medianize

1. Acesse: **https://median.co**
2. Clique em **"Create App"** ou **"Get Started"**
3. Faça login com sua conta (Google, GitHub ou Email)

### 2. Configurar o Aplicativo

1. **App Name**: `Projeto Verão`
2. **App URL**: Cole o link abaixo:
   ```
   https://3002-i0lxt4bhvb0qsg7qrmnma-41bbb83e.us1.manus.computer
   ```
3. **Package Name**: `com.projetoverao.app`
4. **Version**: `1.0.0`

### 3. Configurações Adicionais

**Permissões** (selecione as necessárias):
- ✅ Camera (para captura de fotos)
- ✅ Storage (para galeria de fotos)
- ✅ Internet (para comunicação com servidor)
- ✅ Notifications (para lembretes)

**Aparência**:
- Status Bar Color: `#000000` (preto)
- Navigation Bar Color: `#000000` (preto)
- Orientation: `Portrait` (retrato)

### 4. Ícone e Splash Screen

Se desejar personalizar:

1. **Ícone do App** (192x192 ou 512x512 PNG)
2. **Splash Screen** (imagem de carregamento)

*Dica: Você pode deixar os padrões do Medianize por enquanto*

### 5. Gerar o APK

1. Clique em **"Build"** ou **"Generate APK"**
2. Aguarde o build ser concluído (geralmente 2-5 minutos)
3. Você receberá um link para download do APK

### 6. Instalar no Celular

**Opção A: Via Email**
- Medianize enviará um link por email
- Clique no link no seu celular
- Clique em "Instalar"

**Opção B: Download Manual**
- Baixe o arquivo `.apk`
- Envie para seu celular via WhatsApp, Telegram, etc.
- Abra o arquivo no celular
- Clique em "Instalar"

**Opção C: QR Code**
- Medianize gera um QR code
- Escaneie com seu celular
- Clique em "Instalar"

## 🔄 Atualizar o APK

Se quiser atualizar o app depois:

1. Volte ao Medianize
2. Clique em "Edit" no seu app
3. Modifique as configurações se necessário
4. Clique em "Build" novamente
5. Baixe o novo APK
